**Last Updated:** 2026-06-18

Leave26 is a progress and motivation app for dart players.

This Privacy Policy explains how Leave26 currently handles user data and what information is stored by the application.

---

# 1. Overview

The current MVP version of Leave26 is designed as a local-only application.

No user account is required.

No registration is required.

No personal data is collected by the developer.

No personal data is transmitted to external servers.

The developer has no access to any data stored by the application on the user's device.

---

# 2. Data Collection

Current status:

```text
No personal data is collected by the developer.
```

The current MVP version does not collect:

- names
- email addresses
- phone numbers
- postal addresses
- precise location data
- contacts
- payment information
- health information
- browsing history
- user identifiers

The application does not use third-party tracking technologies.

---

# 3. Local Device Storage

Leave26 stores application data locally on the user's device.

This may include:

- completed checkouts
- checkout history entries
- checkout routes and paths
- 180 counter progress
- goals and milestone progress
- application settings

This information remains on the user's device and is not transmitted to external servers.

---

# 4. Checkout History

Users may manually create checkout history entries.

These entries may contain:

- checkout number
- checkout route
- date and time information

This information is used only to display progress and history within the application.

The developer does not receive or access this information.

---

# 5. User-Generated Content

Leave26 allows users to enter custom checkout routes and notes related to their dart progress.

Any information entered by the user is stored locally on the user's device and is not shared with other users or transmitted to external servers.

The application does not provide public profiles, community features, messaging functionality or content sharing capabilities.

---

# 6. Photos and Media

The current MVP version does not support photo uploads, media uploads or image storage.

If future versions introduce photo or media functionality, this Privacy Policy will be updated before release.

---

# 7. Third-Party Services

The current MVP version does not use third-party services for:

- analytics
- advertising
- cloud storage
- user authentication
- user tracking

The application currently does not use services such as:

- Firebase Analytics
- Firebase Authentication
- Google AdMob
- Google Analytics
- Mixpanel
- Amplitude

---

# 8. Data Sharing

Current status:

```text
No user data is shared with third parties.
```

The application does not sell, rent, share or transfer user data to advertisers, analytics providers or external partners.

---

# 9. Data Deletion

Because all data is stored locally on the user's device, users may delete their data by:

- clearing the application's data through device settings
- uninstalling the application

Future versions may provide additional in-app options for deleting or resetting data.

---

# 10. Children's Privacy

Leave26 is intended for dart players of various age groups.

The application does not knowingly collect personal information from children.

Because no personal data is collected by the developer, Leave26 currently presents a minimal privacy risk for younger users.

---

# 11. Future Features

Future versions of Leave26 may introduce optional features such as:

- Google Sign-In
- Firebase Authentication
- cloud synchronization
- user accounts
- backups
- analytics
- advertising
- premium functionality

If any of these features are implemented, this Privacy Policy will be reviewed and updated before release.

---

# 12. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time as Leave26 evolves.

Any future implementation of cloud services, accounts, analytics, advertising or other data-related functionality will result in an updated version of this document.

The most recent version will always be published with the application.
